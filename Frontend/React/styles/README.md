# Frontend / React / styles

One CSS file per component. All use a dark cinema theme (`#0a0a0f` background, blue accent `#4169E1`).

## Files
| File | Styles for |
|---|---|
| `navbar.css` | Fixed top navbar, logo, nav links, user avatar circle, logout button |
| `hero.css` | Hero banner, overlay gradient, genre category chips, featured movie cards, promo banner section |
| `catalog.css` | Catalog page layout, filter sidebar, search bar, rent/buy toggle switch, product grid cards, pagination |
| `dvddetail.css` | DVD detail page, poster, info panel, credits block, pricing block, rent/buy action buttons |
| `account.css` | Account page, profile card, stats row, current rentals & history sections, settings list |
| `login.css` | Login/register card, mode toggle (Sign In / Create Account), form inputs, error messages |
| `footer.css` | Footer layout and styling |
