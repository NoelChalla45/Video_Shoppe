# Frontend Styles

CSS files are organized by page or feature area.

## Current Style Files
- `login.css`
- `navbar.css`
- `hero.css`
- `catalog.css`
- `dvddetail.css`
- `cart.css`
- `account.css`
- `employee.css`
- `owner.css`
- `footer.css`

## Convention
- One main CSS file per major component/page.
- Shared visual variables come from global styles.
- Role-specific pages have dedicated style files (`employee.css`, `owner.css`).
