# Backend / src

Main server source code.

## Files
- `index.js` — Entry point. Sets up Express, middleware (CORS, JSON), mounts all route files, and starts the server on port 5000.

## Folders
- `routes/` — Individual route files grouped by feature (auth, inventory, rentals, etc.)

> **Status: In Progress** — Server entry point complete. More route files to be added as features are built.
